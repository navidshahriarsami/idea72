## All Pathways from Subclass 500 (Without Relying on 485)

### PATH 1: Direct to 189/190/491 (via ACS Skills Assessment)
- Complete degree → Get ACS skills assessment → Lodge EOI in SkillSelect
- Must meet Australian Study Requirement (2 years CRICOS study in Australia)
- No 485 required IF you can accumulate enough points + have skills assessment
- CRITICAL: You need a bridging visa or a valid visa while EOI is pending
- Most competitive — points threshold 85–95 in 2026

### PATH 2: Subclass 482 (Skills in Demand) → Subclass 186 (ENS)
- Employer sponsors you on 482 Core Skills stream
- After 2 years full-time work (portable across sponsors) → apply 186 (PR)
- Bypasses 485 entirely
- Networking/ICT occupations are on Core Skills Occupation List (CSOL)
- English requirement for 186: IELTS 6.0 overall

### PATH 3: Subclass 485 → 189/190/491 (Traditional)
- 485 gives 2–4 years work rights
- Use that time to get skills assessment + accumulate points + work experience
- Then lodge EOI → receive invitation → apply for PR visa

### PATH 4: Subclass 190 Direct State Nomination
- Some states nominate directly from student/graduate status (no 485 required)
- Requires occupation on state's occupation list
- Adds 5 bonus points to score
- State commitment: live/work in nominating state for 2 years

### PATH 5: Subclass 491 Regional
- 5-year provisional visa
- 15 bonus points vs 190's 5 points — easier to get invited
- After 3 years living/working in regional area → apply Subclass 191 (PR)
- State/territory or eligible relative sponsor required

### PATH 6: National Innovation Visa (NIV)
- For exceptional talent in specific fields
- Check: immi.homeaffairs.gov.au/visas/working-in-australia/visas-for-innovation

- Month 0–3: ACS skills assessment lodgement + result
- Month 3–6: Lodge EOI in SkillSelect
